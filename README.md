# 日本語 — App de estudo de Japonês

PWA pessoal para estudar japonês (offline-first, instalável na tela de início).

## Funcionalidades
- **Estudar** — flashcards com repetição espaçada (SRS)
- **Kanji (Kanjidex)** — consulta, ordem dos traços animada, prática de escrita com o dedo, quizzes de significado/leitura (com modo difícil e modo "palavras")
- **Treino** — drills de te-form, contadores, adjetivos; listening (TTS + voz nativa); leitura com furigana e lookup de palavra por toque; compor frases (romaji/kana, por nível)
- **Perfil RPG** — níveis, atributos, radar, conquistas, missões diárias, personalização
- Funciona **offline**, com auto-atualização.

## Como usar
Acesse a página publicada (GitHub Pages). No iPhone: abra no Safari → Compartilhar → **Adicionar à Tela de Início**.
